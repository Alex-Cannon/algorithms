#### { Instructions }
Rewrite the myConcat function which appends contents of arr2 to arr1 so that the function uses arrow function syntax.

#### { My Solution }
```
const myConcat = (arr1, arr2)=> {
  "use strict";
  return arr1.concat(arr2);
};
```
